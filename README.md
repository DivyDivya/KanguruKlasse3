# 🦘 Kanguru Math Practice App

An interactive web application to help Class 3 students (ages 8-9) prepare for the Kangaroo Math Contest!

## Features

✨ **Practice Tests**: Generate 30-question practice tests with mixed difficulty levels
📚 **Review Mode**: Practice questions you previously got wrong for targeted learning
📊 **Track Progress**: View your test history and see your improvement over time
💡 **Learn from Mistakes**: Get detailed explanations for every answer
🎯 **Points System**: Earn 3-5 points per question based on difficulty

## How to Use

### Starting the App

1. **Open the app**: Simply double-click `index.html` to open it in your web browser
   - Or drag and drop `index.html` into your browser window
   - No internet connection needed - works completely offline!

2. **Choose what to do**:
   - **Start New Practice Test**: Take a fresh 30-question quiz
   - **Review Wrong Answers**: Practice questions you got wrong before
   - **View Past Tests**: See your test history and track progress

### Taking a Test

1. Read each question carefully
2. Click on your answer choice (A, B, C, D, or E)
3. Click "Submit Answer" to check if you're correct
4. Read the explanation to understand the answer
5. Click "Next Question" to continue
6. Complete all 30 questions

### Understanding Your Score

- **Easy Questions**: 3 points each (10 questions)
- **Medium Questions**: 4 points each (10 questions)
- **Hard Questions**: 5 points each (10 questions)
- **Maximum Score**: 120 points per test

### Score Ratings

- **90%+ (108+ points)**: Outstanding! ⭐⭐⭐
- **75-89% (90-107 points)**: Great job! ⭐⭐
- **60-74% (72-89 points)**: Good effort! ⭐
- **Below 60%**: Keep practicing! 💪

## Question Topics

The app covers Class 3 math topics including:

- **Arithmetic**: Addition, subtraction, multiplication, division
- **Patterns**: Number sequences and patterns
- **Geometry**: Shapes, perimeter, area
- **Logic**: Word problems and puzzles
- **Measurement**: Time, length, weight, volume

🎯 **Read Carefully**: Make sure you understand what the question is asking
🤔 **Think It Through**: Don't rush - take your time on each question
📝 **Learn from Mistakes**: Read the explanations even when you get it right
🔄 **Practice Regularly**: Take a test every few days to improve
📊 **Track Progress**: Check your history to see how you're improving

## For Parents and Teachers

### Progress Tracking

The app automatically saves:
- All completed tests with scores and timestamps
- Questions answered incorrectly (for targeted practice)
- Time taken for each test
- Detailed breakdown of correct/incorrect answers

### Clearing Data

To clear all test history:
1. Open your browser's developer console (F12)
2. Type: `localStorage.clear()`
3. Press Enter
4. Refresh the page

### Browser Compatibility

- **Recommended**: Google Chrome, Microsoft Edge, Mozilla Firefox, Safari
- **Minimum**: Any browser with HTML5, CSS3, and ES6 JavaScript support
- **Mobile**: Works on tablets and phones (responsive design)

## Troubleshooting

**Q: My test history disappeared**
- Test history is stored in browser localStorage. Clearing browser data will erase it.
- Use the same browser and don't clear browser data to keep history.

**Q: The app won't open**
- Make sure you're opening index.html in a web browser (not a text editor)
- Try a different browser if one doesn't work
- Ensure JavaScript is enabled in your browser

**Q: I can't click the Submit button**
- You need to select an answer first before the Submit button becomes active
- Click on one of the answer options (A, B, C, D, or E)

## About the Kangaroo Math Contest

The Kanguru (Kangaroo) Math Contest is an international mathematics competition for students of all ages. It focuses on:
- Problem-solving skills
- Logical thinking
- Mathematical reasoning
- Creative approaches to math

This practice app helps Class 3 students prepare by providing similar types of questions with helpful explanations.
