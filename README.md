# 🦘 Känguru Mathe-Übung

Interactive math practice app for the Känguru der Mathematik contest (Class 3/4).

## Features
- 24-question format with official Känguru scoring
- Bilingual (German/English)
- 60+ practice questions
- Colorful, kid-friendly design

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub
2. Upload all files from this folder
3. Go to Settings → Pages
4. Select "Deploy from main branch"
5. Your app will be live at: `https://YOUR-USERNAME.github.io/REPO-NAME/`

## Files
- `index.html` - Main application
- `app.js` - Logic and scoring
- `questions-data.js` - Question bank
- `styles.css` - Styling
